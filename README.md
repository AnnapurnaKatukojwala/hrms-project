# 🤖 AI-Powered Human Resource Management System (HRMS)

An AI-powered full-stack Human Resource Management System that helps organizations manage employees, leaves, and HR-related queries using a smart chatbot and analytics dashboard.

---

## 📌 Project Overview

This project is a **full-stack HRMS application** that combines traditional HR management features with AI capabilities.  
It allows HR teams to manage employees, handle leave requests, and interact with an AI chatbot for quick HR assistance.

The system is built using a **React frontend + FastAPI backend** and demonstrates real-world integration of APIs and AI features.

---

## ✨ Features

### 👨‍💼 Employee Management
- Add employees
- View employee list
- Auto-generated employee bio
- Department-wise organization

### 🏖️ Leave Management
- Apply for leave
- View leave requests
- Approve / Reject leaves
- Status tracking (Pending / Approved / Rejected)

### 🤖 AI Chatbot
- HR-related question answering
- Employee count and basic queries
- Fallback response for unknown questions

### 📊 Analytics Dashboard
- Total employee count
- Department-wise distribution
